长沙口岸介绍

1904年（光绪三十年）7月长沙正式开埠，属全国80处“约开商埠”之一，成为我省最主要的对外通商口岸。长沙现有一个国家一类航空口岸即黄花国际机场航空口岸，等级为4E级，是我国十大区域性国际航空枢纽之一；两个二类口岸，长沙霞凝水运口岸和长沙霞凝铁路口岸，其中霞凝水运口岸是长沙及其周边地区唯一的大型综合性港口，承担着长沙及湘中地区60%以上的内支线集装箱装卸任务；霞凝铁路口岸开通了长沙至深圳的“五定班列”和长沙至中亚、长沙至俄罗斯、长沙至欧洲的中欧班列；一个保税物流中心（B型）——长沙金霞保税物流中心，规划面积1500亩，总投资约15亿元人民币，2009年正式封关运行，是湖南扩大对外开放的重要标志；一个国际邮件监管中心——长沙邮政国际快件监管中心，是我省唯一的国际邮件互换局兼交换站，负责长沙海关关区范围内进出口国际邮件的接收、转发及指定进出口国际邮件总包的接收、封发；三个出口烟花海关监管点，其中湖南华洋仓储已建成运营，是我市第一家出口烟花海关监管场所，景美仓储、速福仓码已通过海关验收并对外开展业务。正在建设的有湖南电子口岸（长沙）信息平台、长沙黄花综合保税区、长沙跨境贸易电子商务服务平台。全市进出口市场覆盖167个国家和地区，初步形成了一、二类口岸齐全、客货运功能完善、水陆空运输方式配套，与我市开放型经济发展基本相适应的立体口岸开放体系。
